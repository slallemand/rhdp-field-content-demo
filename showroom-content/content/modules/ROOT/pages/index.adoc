== Vue d'ensemble

Bienvenue dans cette démonstration sur la création et le déploiement de contenu personnalisé sur Red Hat Demo Platform (RHDP).

[.lead]
*Public cible:* Solution Architects

[.lead]
*Durée:* 20 minutes

== Scénario métier

Les Solution Architects ont besoin de créer et déployer rapidement des labs et démos personnalisées sur Red Hat Demo Platform pour répondre aux besoins spécifiques de leurs clients.

Cette démonstration illustre comment RHDP simplifie ce processus grâce à:

* *Provisionnement rapide* - Clusters OpenShift multi-cloud en minutes avec CNV
* *Automatisation du déploiement* - Field Source Content déploie via GitOps (Ansible/Helm)
* *Réutilisation et partage* - Skills Marketplace pour étendre vos démos
* *Simplicité GitOps* - Un repo Git = déploiement automatique
